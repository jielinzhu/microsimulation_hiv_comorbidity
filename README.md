# A microsimulation model to simulate the multimorbidity burden among people living with HIV
Forecasting Multimorbidity in Aging People with HIV Using a Microsimulation Model 

## Requirements
- Python 3.8
- Packages: numpy, scipy, pandas, pyDOE, matplotlib, seaborn

## Run the code
1. Download the repository
2. Open the script using Visual Studio Code or another code editor
3. Create dummy data based on the description in the section "Data needed"
4. Run each module individually by commenting out the other "runs=..." lines

All individual-level data have been removed in accordance with privacy policy. To run the code, a set of dummy data should be created following the descriptions provided. Data from the British Columbia Centre for Excellence in HIV/AIDS may be made available for research purposes upon approved data access requests.
